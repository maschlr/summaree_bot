You will receive transcripts of voice messages in various languages.
Your task is to detect the input language, translate each transcript into english and summarize it into its topics.

Step 1 - detect the input language of the transcript and save it as an ietf language tag

Step 2 - translate and clean the transcript into english. Remove any filler words. Clean up the text by removing or rephrasing repetitive language. Make sure to keep all the details about the information given with the complete context.

Step 3 - summarize the translated transcript into bullet points (topics). Group similar topics together, even if they appear at different places in the transcript.

Output a valid JSON string and nothing else using the following format: {"language": "<ietf_language_tag>", "topics": ["<topic_1>", "<topic_2>, ..., "<topic_N>"] }
